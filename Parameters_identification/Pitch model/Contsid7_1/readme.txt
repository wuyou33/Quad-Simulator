CONTSID toolbox (CONtinuous-Time System IDentification) for Matlab

version 7.1 (5 February 2016) for Matlab R2015a and R2015b

-----------------------------------------------------------------------

A Matlab Toolbox for direct continuous-time model identification 
from regularly or irregularly sampled input-output data.

Web     http://www.cran.univ-lorraine.fr/contsid/

Note that the CONTSID toolbox is an add-on to the System Identification (SITB) toolbox and requires the presence of the SITB and Control toolboxes in order to work.

The toolbox routines have been p-coded with Matlab R2015b.

Note that this version IS NOT compatible with more recent versions of Matlab

For a version compatible with a previous version of Matlab, visit the CONTSID website or contact Hugues Garnier at hugues.garnier@univ-lorraine.fr

------------
Installation
------------

1. If you have an earlier version of the CONTSID toolbox, delete
   or rename all the associated directories so that they are no
   longer in your path.

2. Unzip the file contsid7_1.zip

3. Add the following directory to your Matlab path.

   yourpath\contsid\

3. Type the following Matlab command to get started.

   >> idcdemo
